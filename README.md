# Indoor Air Quality Monitoring System using IoT and Blynk Server

## About
A real-time IoT-based indoor air quality monitoring system that tracks 
environmental parameters and sends live data to a mobile dashboard.

## Technologies Used
- ESP32 Microcontroller
- Blynk IoT Cloud Server
- Gas Sensors (MQ-135 / MQ-2)
- Python
- Arduino IDE

## Features
- Real-time air quality monitoring
- Live data on Blynk mobile dashboard
- Threshold-based alerts on mobile
- Full IoT pipeline: sensor → ESP32 → cloud → mobile app

## Circuit Components
- ESP32 development board
- MQ-135 air quality sensor
- Gas sensor
- Connecting wires and breadboard
